# Online Voting System

A simple Python console-based online voting system that allows users to register, log in, and cast their votes. The admin can manage candidates and view the voting results.

## Features
- User registration
- User login authentication
- Cast vote
- Prevent multiple votes from the same user
- Admin panel for managing candidates and viewing results
