# Problem Statement
This project aims to create a simple and secure Online Voting System where users can register, log in, and cast their votes in an election. The system ensures that each user can vote only once and provides an admin panel to manage candidates and view results.

# Scope of the Project
The project focuses on building a basic text-based voting system using Python. It includes user registration, authentication, vote casting, and result display. The system is designed for learning purposes and does not use advanced encryption or real-time networking.

# Target Users
- Students who want to learn basic Python project structure
- Administrators who manage voting data
- Users who wish to participate in the voting process

# High-Level Features
- User registration and login
- Storing user and voting data in JSON files
- Casting a vote
- Preventing multiple votes from the same user
- Admin functions to add candidates and view results
